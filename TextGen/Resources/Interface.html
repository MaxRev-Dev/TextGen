About

Програма генерує HTML-файл з питань у текстовому документі. При редагуванні конфігурації або темплейту будьте уважні. А для редагування дизайну темплейту потрібні базові знання HTML & CSS & Bootstrap 4.

ОСОБЛИВОСТІ

У випадку "вильотів" чи зависань може бути не правильно налаштований конфіг чи темплейт.

КОНЦЕПТ

Назви файлів можна змінити в конфігурації та й не забуваємо про темплейт. Це два слеші (//) - для заголовка. Фігурні дужки ( { } ) - для блоку питань. Тильда (~) - для звичайних відповідей. Та відсоток (%) - для правильних (виділених).

Також у вже існуючу папку згенерувати файли не вийде. Це для того, щоб не було дублікатів та сміття. Та й кирилицею користуватись не рекомендую, просто файли не знайдуться)
